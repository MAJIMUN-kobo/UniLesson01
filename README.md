## 作り方

### プロジェクトを作る
新規プロジェクトを作ります。
・テンプレート：Universal 2D
・プロジェクト名：UniLesson01
・保存場所を決めて下さい。
・プロジェクトを作成

Unityのエディター画面が開いたら準備完了です

### 画面の設定
ゲーム画面のアスペクト比を「16:9」に設定します。

### フォルダの作成
「プロジェクトメニュー」⇒「Create」⇒「Folder」から以下のフォルダを作成します。
・Audio
・Materials
・Scripts
・Texture
